# 《末日进化：基因王座》

## 世界观设定

### 背景：大灾变纪元

2045 年，"红雨"降临地球。一场来自宇宙深处的基因病毒随着红色雨水席卷全球，90% 的人类在 72 小时内发生基因崩溃，变成嗜血的"堕落者"。

幸存者们发现，红雨不仅带来毁灭，也带来了进化。少数人觉醒了"基因锁"，获得了超越常人的能力。

### 力量体系：基因锁九阶

| 阶位 | 名称 | 能力特征 |
|------|------|----------|
| 一阶 | 觉醒者 | 基础身体素质提升 3-5 倍 |
| 二阶 | 强化者 | 觉醒一项基因异能 |
| 三阶 | 掌控者 | 异能成型，可外放 |
| 四阶 | 领主级 | 形成基因领域，碾压低阶 |
| 五阶 | 君王级 | 可改变局部环境规则 |
| 六阶 | 史诗级 | 一人敌城，传说存在 |
| 七阶 | 神话级 | 近乎神明，千年不出 |
| 八阶 | 传奇级 | 只存在于古籍中 |
| 九阶 | 基因王座 | 唯一神，掌控进化法则 |

### 世界格局

**三大安全区：**
- **北境钢铁城** - 军事化统治，崇尚武力
- **南海联盟** - 商业帝国，科技发达
- **西荒自由港** - 混乱之地，强者为尊

**四大禁区：**
- **红雨源头·陨石坑** - 传说中有终极进化秘密
- **堕落者巢穴·地下都市** - 堕落者王者的领地
- **变异兽森林·死亡禁区** - 高阶变异兽栖息地
- **基因风暴海** - 能量紊乱，九死一生

### 核心设定

**基因药剂：** 从堕落者和变异兽体内提取基因碎片，可制作进化药剂，但有失控风险。

**进化点：** 击杀堕落者/变异兽可获得，用于突破基因锁瓶颈。

**基因共鸣：** 高阶强者可与异性产生基因共鸣，双修可加速进化（爽点设定）。

**天命系统：** 主角独有的进化辅助系统，可解析基因、优化功法、预测危险。

## 主角设定

### 姓名：秦烈

**身份：** 前特种部队教官 → 末日最强进化者

**年龄：** 28 岁

**外貌：** 
- 身高 188cm，身材完美，肌肉线条分明
- 棱角分明的脸庞，眼神锐利如刀
- 黑色短发，左眉骨有一道疤痕（增添野性魅力）
- 常穿黑色作战服，气场强大

**性格：**
- 杀伐果断，不圣母
- 对敌人冷酷，对自己人护短
- 有仇必报，有恩必还
- 情感上主动出击，不矫情

**金手指：天命进化系统**
- 可解析任何基因序列
- 自动优化修炼功法
- 危险预知能力
- 基因融合无副作用
- 可共享进化点给伴侣（双修增益）

**初始异能：** 【雷霆掌控】+【基因吞噬】（可吞噬他人基因能力据为己有）

**成长路线：** 从底层幸存者一路杀上基因王座，建立末日帝国，坐拥多位绝世佳人。

## 爽点设计

### 战斗爽点
- 越阶挑战，以弱胜强
- 一人单挑整个势力
- 关键时刻突破，反杀打脸
- 吞噬敌人能力，越来越强

### 情感爽点
- 高冷女总裁主动倒追
- 萝莉杀手变身小迷妹
- 女战神被打败后乖乖听话
- 医仙温柔专属照顾
- 堕落者女王献上一切

### 双修设定（基因共鸣）
- 与女主双修可加速进化
- 每次双修后实力大涨
- 女主们主动要求"帮助修炼"
- 描写要暧昧但不低俗

---

## 写作风格

- **节奏：** 快节奏，3 章一个小高潮，10 章一个大高潮
- **对话：** 简洁有力，不拖沓
- **战斗：** 详细描写，突出主角的强大
- **情感：** 主动直接，不扭捏
- **尺度：** 暧昧撩拨为主，关键场景适当露骨
