# 📦 自动部署指南

## 新增章节后，只需两步：

### 1️⃣ 在 index.html 中添加章节配置

打开 `index.html`，找到 `chapters` 数组，添加新章节：

```javascript
const chapters = [
    { id: 1, file: 'chapters/001-红雨之夜.md', title: '第一章：红雨之夜，她敲开了我的门' },
    { id: 2, file: 'chapters/002-雷霆觉醒.md', title: '第二章：雷霆觉醒，她的身体在发光' },
    // ... 现有章节
    { id: 7, file: 'chapters/007-新章节.md', title: '第七章：新章节标题' }  // ← 新增这行
];
```

### 2️⃣ 运行部署脚本

```bash
./deploy.sh
```

**完成！** GitHub Pages 会在 1-2 分钟内自动更新！

---

## 原理说明

- 页面使用 `marked.js` 库自动解析 Markdown
- 点击章节时，通过 `fetch()` 动态加载对应的 `.md` 文件
- **不需要**手动复制内容到 HTML
- **只需要**在 `chapters` 数组里添加配置

---

## 以后写小说的流程

1. 写新章节 → 保存为 `chapters/007-xxx.md`
2. 在 `index.html` 的 `chapters` 数组添加一行配置
3. 运行 `./deploy.sh`
4. 手机刷新页面，自动看到新章节！
