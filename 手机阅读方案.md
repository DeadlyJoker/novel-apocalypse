# 手机阅读方案

## 方案一：Telegram 频道推送（推荐）

**优势：**
- 手机原生体验，阅读流畅
- 支持富文本格式
- 可离线缓存
- 推送通知提醒更新

**配置步骤：**
1. 创建 Telegram 频道
2. 在 OpenClaw 配置中绑定频道 ID
3. 每章完成后自动推送

**命令：**
```bash
# 发送到 Telegram
npx openclaw message send --channel telegram --target "@novel-channel" --file chapter.md
```

---

## 方案二：GitHub + 静态页面

**优势：**
- 永久存储，可分享链接
- 支持 Markdown 渲染
- 多设备同步

**步骤：**
1. 创建 GitHub 仓库
2. 启用 GitHub Pages
3. 每章 commit 后自动部署

**仓库结构：**
```
novel-apocalypse/
├── README.md          # 目录 + 简介
├── chapters/
│   ├── 001.md
│   ├── 002.md
│   └── ...
└── assets/            # 封面图等
```

**手机访问：** `https://yourname.github.io/novel-apocalypse`

---

## 方案三：Obsidian 同步

**优势：**
- 本地优先，隐私好
- Obsidian Mobile 体验优秀
- 支持 iCloud/坚果云同步

**步骤：**
1. 将小说目录添加到 Obsidian
2. 开启 Obsidian Sync 或配置第三方同步
3. 手机端安装 Obsidian App

---

**优势：**

**步骤：**
